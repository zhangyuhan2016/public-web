# IDE

  • vs code (opens new window)

    微软出品，开源，基于Electron开发，体积小，扩展多。

  • WebStorm Eap (opens new window)

    有强大的智能补全和提示，写起来比 vs code 舒服很多。 但是是一款收费软件，这里建议使用免费的 WebStorm Eap 版，是预发布版本，每月更新体验测试，缺点可能不稳定。

    • WebStorm 中文语言包 (opens new window)

# Node

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。

  • node (opens new window) 官网下载最新稳定版

  • nvm (opens new window) 使用nvm管理和切换Node版本

  • npx (opens new window) 使用npx也可以用来切换Node版本

    npx node@4 -e "console.log(process.version)"
npx node@6 -e "console.log(process.version)"

# npm & yarn

使用淘宝镜像加速模块下载

  • npm

    npm config set registry https://registry.npm.taobao.org --global
npm config set disturl https://npm.taobao.org/dist --global
    • 你不知道的 npm (opens new window)

  • yarn (opens new window)

    npm install -g yarn
yarn config set registry https://registry.npm.taobao.org --global
yarn config set disturl https://npm.taobao.org/dist --global

    用yarn add某第三方库名代替npm install某第三方库名。
